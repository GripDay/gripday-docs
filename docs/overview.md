# Project Overview

**GripDay** is a revolutionary open-core B2B marketing automation platform that addresses a $6.1B market opportunity with 12.8% CAGR growth. Built with cutting-edge Java 21 and React 19 technologies, it delivers enterprise-grade capabilities through a sustainable open-core business model.

## 🎯 Mission Statement

The **first enterprise-grade open-core B2B marketing automation platform** that enables sustainable SaaS business growth through modular architecture - combining free core modules with commercial premium features.

## 🏗️ Architecture Overview

GripDay implements a **microservices-first architecture** designed for enterprise-scale marketing automation:

### Core Microservices (MVP Phase 1)

1. **User Service** - Centralized auth with JWT and RBAC
2. **API Gateway** - Intelligent routing, rate limiting, and security
3. **Contact Management Service** - Contact and company lifecycle management
4. **Email Marketing Service** - Email creation, delivery, and tracking
5. **Campaign Automation Service** - Workflow engine and automation
6. **Form Builder Service** - Form creation and submission processing
7. **Scoring Service** - Lead scoring and qualification engine
8. **Analytics Service** - Reporting and business intelligence

### Technology Stack

- **Backend**: Java 21, Spring Boot 3, Spring Security 6, Apache Kafka
- **Frontend**: React 19, TypeScript, Mantine UI, Vite
- **Database**: PostgreSQL 16+ with CloudNativePG operator
- **Infrastructure**: Kubernetes, Istio service mesh, Docker
- **Observability**: Prometheus, Grafana, Jaeger, ELK Stack
